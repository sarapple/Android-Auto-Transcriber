Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app\src\main\AndroidManifest.xml
* assets\ => app\src\main\assets\
* res\ => app\src\main\res\
* src\ => app\src\main\java\
* src\com\midisheetmusic\Changelog.txt => app\src\main\resources\com\midisheetmusic\Changelog.txt
* src\com\midisheetmusic\FileUri.java.orig => app\src\main\resources\com\midisheetmusic\FileUri.java.orig
* src\com\midisheetmusic\sharedpreferences.txt => app\src\main\resources\com\midisheetmusic\sharedpreferences.txt
